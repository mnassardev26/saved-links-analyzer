# Privacy Policy

Saved Links Analyzer is local-first.

- Saved Facebook item metadata is stored in Chrome extension local storage.
- The extension does not sell, share, or transmit saved-link data to a backend.
- The extension only reads content visible to the signed-in user on `facebook.com/saved`.
- Exported CSV files are created locally by the browser at the user's request.
- You can clear scanned data from the extension settings panel.
